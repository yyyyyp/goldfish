# 


## 框架

* JS+Flask+MongoDB 



## 安装

* 通过<code>pip install -r requirements</code>安装所需包  


## 运行

* 命令行键入“python server.py runserver”启动服务器  
* 浏览器访问“127.0.0.1:5000 ”，点击“请选择”按钮，选择文件并上传  
* 可以起一个虚拟机测试，虚拟机采用桥接模式.

## 流程

* 进入网页点击upload按钮
* 上传文件至服务器端，上传成功后跑模型
* 此处省略
* 生成新的视频，放入指定文件夹内，并将数据保存进数据库
* 前端接收信息，并播放视频
* 查看表格（数据库内容
